what is bioaurora?

      A transcriptome refers to a collection of all transcripts within a cell under a physiological condition. With the continuous development and improvement of high-throughput sequencing, RNA-Seq has become one of the important methods for studying biological problems.
      We used lung cancer tissue data as a sample, and performed transcriptome data such as quality control, filtration and other preprocessing, data comparison, gene expression differential analysis, variable shear analysis, fusion gene analysis, GO analysis and other steps. Although there are nearly 100 kinds of tools for completing the above steps, the data processing capabilities of these tools have their advantages and disadvantages, and the degree of fit between the tools is also different. We compared various software and programs and found the most suitable tool for the transcriptome of human tissues. Finally, we chose Trimmomatic, Tophat, cufflink, deseq, and ASprofile as analysis tools.

what is new in bioaurora?

      Using the glue properties of python, the tedious RNA-Seq analysis process is integrated into a system, including tool integration, file format docking, results analysis, and more. The user can set several parameters, submit tasks, and automatically process in the background. The user can be notified after the results run, and the results can be viewed with the complete log file. Constructing such an intelligent automated analysis system can minimize the cumbersome and repetitive tasks brought about by big data analysis and give convenience to other researchers.
